# 🏫 Nearby Schools API - Educase Internship Assignment

This is a Node.js + Express + MySQL backend project built as part of an internship assignment for **Educase**. The API allows users to register schools and find nearby schools based on their current location using latitude and longitude.

---

## 🚀 Features

- Register a new school with name, address, and geo-coordinates.
- List all schools sorted by proximity to the user.
- Filter schools within a 5 km radius.
- Input validation using **Zod**.
- Modular structure with controllers, routes, and models.
- Built with **TypeScript** and **MySQL**.

---

## 📦 Tech Stack

- Node.js
- Express.js
- TypeScript
- MySQL
- Zod (for schema validation)

---

## 🛠️Setup Instructions
- git clone https://github.com/SamayMody/educase.git
- cd educase/server
- npm install
- update .env by your credentials
- npm start to start the server
